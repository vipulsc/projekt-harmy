## ‚ö° Quick Start

1. **Clone & Install**
   ```bash
   git clone https://github.com/vipulsc/projekt-harmy.git
   cd harmy
   npm install
   ```
2. **Set Environment**
   - Copy `.env.example` ‚Üí `.env.local`
   - Add your Clerk keys
3. **Run**
   ```bash
   npm run dev
   ```
